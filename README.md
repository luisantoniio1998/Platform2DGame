# Platform2DGame

2D Platform Game developed for Multimedia Systems Course evaluation using Unity and C#.

## Technologies
- Unity Engine
- C# Programming
- 2D Sprites/Animation
- Unity Physics2D

## Features
- Platform mechanics
- Character controller
- Enemy AI
- Level design
- Score system

## Setup
1. Unity 2D installed
2. Clone repository
3. Open in Unity
4. Build/Run game

## Project Structure
Assets/
├── Scripts/
├── Sprites/
├── Animations/
├── Scenes/
└── Prefabs/

## Controls
- Arrow Keys: Movement
- Space: Jump
